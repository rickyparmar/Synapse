\# Synapse



Synapse is an AI-powered mental wellness and cognitive health platform designed to support dementia care and student mental well-being. The platform combines AI voice therapy, cognitive training games, EEG analysis, caregiver monitoring, VR-based peer support, and personalized mental health planning.



\## Features



\* AI Voice Therapist powered by ElevenLabs

\* Cognitive Training Games

\* EEG Data Analysis and PDF Report Generation

\* Caregiver Dashboard

\* Mental Health Planner

\* VR Peer Support Environment

\* Multilingual Resource Hub

\* Accessibility Features including Dyslexia-Friendly Fonts



\## Tech Stack



\* Next.js 15

\* React 19

\* Tailwind CSS

\* Supabase

\* ElevenLabs SDK

\* Python EEG Processing Backend



\## Installation



Clone the repository:



```bash

git clone <repository-url>

cd synapse

```



Install dependencies:



```bash

npm install

```



Run the development server:



```bash

npm run dev

```



Open http://localhost:3000 in your browser.
